My projects

geo arsh

my first ever project, built for an association to help them manage their community and events. started with pure html and css, then integrated django framework for the backend functionality. used python for server-side logic and javascript for interactive features on the frontend.

phsycologos

created for the psychology association to showcase their work and connect with students. built with next.js for fast page loads and better seo, typescript for cleaner code, and tailwind for styling. deployed on vercel for seamless updates.

meli.sh

this website! my personal portfolio where i share my projects, essays, and thoughts. features real-time visitor tracking so you can see how many people are browsing, and a comment system where visitors can leave their thoughts. built using node.js for the websocket backend and pure html/css for full control over the design.

lime

upcoming startup with lieh, focused on helping starter bands and independent artists get discovered and build their audience. the platform will connect musicians with listeners who actually care about finding new music. built with typescript and tailwind for the frontend experience, and django backend with python for handling user data and music recommendations.
